# Airline Interface Web Application

## Overview
This project is a **full-stack airline interface** built with **SQL, Python (Flask), HTML, and CSS**.  
It provides different login portals for **airline staff** and **customers**, each with tailored functionalities.

## Features
- **User authentication** with role-based access (staff vs. customers)
- Customers can:
  - Login and view their profile
  - Book flights and manage reservations
- Staff can:
  - Login to view passenger data
  - Manage flights, schedules, and reservations
- Database-backed airline system with persistent storage

## Technologies
- Python (Flask framework)
- SQL (database management)
- HTML, CSS (frontend design)
